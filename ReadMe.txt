1. はじめに
　この度は、「クリスマス中止のお知らせ。」をダウンロードしていただき
ありがとうございます。本書では、ゲームの概要、操作法について説明いた
します。

2. 概要
　空を飛んでいるサンタクロースを大砲で3回 命中させれば、ゲームクリア
です。途中、プレゼントが降ってきますが、避けてください。大砲にプレゼ
ントが3回当たるとゲームオーバーです。
　サンタに命中させるまでの時間、命中したプレゼントの数で点数が決まり
ます。
　レベルは3つあり、初級はサンタも大砲もプレゼントもゆっくり動きます。

3. 操作方法
 ←→ : 大砲の移動
 ↑↓ : 大砲の向きの変更
SPACE : 大砲を打つ
